## Setting Up a Test Account

### Option 1: Create Test Account in App Store Connect (Recommended)

1. **Go to App Store Connect**
   - Navigate to: https://appstoreconnect.apple.com
   - Sign in with your Apple Developer account

2. **Create Test User**
   - Go to **Users and Access** → **TestFlight** (or **App Store Connect Users**)
   - Click **+** to add a new user
   - Create a test user with:
     - Email: `test.astromatch@yourdomain.com` (or use a Gmail/email you control)
     - Role: **App Manager** or **TestFlight Beta Tester**
     - Note: For review purposes, create a real account that Apple reviewers can use

3. **OR: Use Your Own Account**
   - You can use your own email/password if you want reviewers to test with a real account
   - Make sure the account has sample profiles and matches set up

### Option 2: Create Test Account in Your App

1. **Before Submission:**
   - Sign up in your app with a dedicated test email (e.g., `test@astromatch.app`)
   - Create a complete profile:
     - First name
     - Birth date
     - Location
     - Profile photos
     - About me section

2. **Ensure Test Data Exists:**
   - Make sure there are some matches/compatible profiles in the database
   - Test that the account can see match scores and labels
   - Verify all features work correctly

3. **Use This Account Info in Review Notes:**
   ```
   Email: test@astromatch.app (or your test email)
   Password: [secure password you create]
   ```

### Option 3: Use TestFlight Internal Testing

1. **Set Up Internal Testers:**
   - In App Store Connect → **TestFlight**
   - Add internal testers (up to 100 with Apple Developer account)
   - Create accounts for testing

2. **For Review:**
   - Provide credentials for an account that's already set up in your production/test environment

# Android Studio Build Options

## Common Build Menu Items:
- **Build** → **Make Project** (may be called "Make Module 'app'" or just "Make")
- **Build** → **Rebuild Project**
- **Build** → **Clean Project**
- **Build** → **Build Bundle(s) / APK(s)** → **Build Bundle(s)**

## If "Make Project" is Missing:
You can:
1. **Just run the app directly** - Android Studio will build automatically
2. Look for: **Build** → **Make Module 'app'**
3. Or use: **Build** → **Rebuild Project** (slower but thorough)

## Recommended: Just Run!
You don't actually need to build separately - just:
1. Click the **Run** button (green play icon ▶️)
2. Or press `Shift+F10` (Mac: `Ctrl+R`)
3. Android Studio will build automatically before running

## Quick Test:
- Connect device/emulator
- Click Run button
- That's it! It will build and install automatically
